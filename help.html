Quick Help Guide

Logging into SpiraTeam

  1. Enter your SpiraTeam URL

    Please enter the web address that you use to access SpiraTeam® in your browser. This is usually of the form 'http://(hostname)/SpiraTeam'. Make sure that you remove any suffixes from the address (e.g. Default.aspx)

  2. Enter your user name

    Please enter the user name you use to log in to SpiraTeam. (Not Case Sensitive)

  3. Enter your RSS token

    Please enter your RSS token including the curly braces i.e {ExampleRSS}

User Interface and Buttons

  • Help

    This button will open an in app help menu with basic functionality information.

  • Disconnect

    This button will close your connection with SpiraTeam and clear your data. WARNING: your spreadsheet data will be erased as a security measure on disconnect.

  • Clear

    This button will clear the contents of your current tab including the template.

  • Projects

    Select a project from the menu.

  • Artifacts

    Select an artifact from the menu.

  • Load Template

    Click this button to generate the template for the project and artifact you’ve selected.

  • Save

    Clicking this button will save your current spreadsheet information into a new tab.

  • Send to SpiraTeam

    This button will send your properly formatted data to SpiraTeam and it will be added to the current project.

  • Import to SpiraTeam

    Not Currently Supported

Requirement Standard Field values

  • Requirement Id

    Stores the id of the item. This field should be left blank when sending data to SpiraTeam

  • Requirement Name

    **Required Field** The name of the item. This field supports indentation, add a “ > “ symbol to indicate how the items in the artifact hierarchy are organized. 
               	Example:
            Item 1
            >Item 2 child of item 1
            >Item 3 child of item 1
            >>Item 4 child of item 3

  • Requirement Description

    Long description of the item. Rich text is available through HTML tags.

  • Release Version

    Select a release/iteration from the menu.

  • Type

    **Required Field** Select the type from the menu.

  • Importance

    Assign the importance/priority from the menu.

  • Status

    Select the status from the menu.

  • Estimate

    The estimate in points for the item

  • Author

    Select a user from the menu.

  • Owner

    Select a user from the menu.

  • Component

    Select a component from the menu.

  • Linked Requirements

    Not Currently Supported

  • Comment

    Not Currently Supported

Custom Field Entries

  • Text

    Enter any text. Include HTML tags for rich text.

  • Decimal

    Enter any positive number.

  • Integer

    Enter a non-decimal number.

  • Date

    Enter the date in any standard format.

  • List

    Select a value from the dropdown list.

  • MultiList

    Select an item from the provided list. This is not currently supported

  • User

    Select a name from the list of users
